How to Adjust Course Settings, Extend Access and Manage Groups
  1. Configure Course Settings
    Customize how learners interact with your course:
    • Course Access Settings: Control open enrollment, password-protected access, or payment-based entry.
    • Course Navigation Settings: Choose between linear progression or free navigation.
    • Display and Content Options: Adjust layout preferences, certificates, materials, etc.
    • Course Users: Assign and manage course participants.
    • Share Course Settings: Add instructors for co-teaching.
  2. Configure Course Access Settings
    • Access Mode: Open ‚Äì Anyone can access freely.
    • Buy Now: One-time payment via PayPal/Stripe
    • Recurring: Subscription-based access
    • Closed: Manual/group enrollment only
  3. Course Navigation Settings
    • Linear: Sequential lesson completion required
    • Free Form: Learners access any lesson freely
  4. Customize Display and Content Options
    • Enable downloadable course material and certificates
    • Restrict course content visibility to enrollees
    • Optional challenge exams and pagination settings
  5. Assign Course Users
    • Manually add or remove users using the dual-column interface
    • Group-managed users are handled separately in the Group Admin area
  6. Share Course Settings
    • Allow co-instructors to access and manage the course
    • Useful for team teaching, moderators, or assistants
  7. How to Extend Course Access
    • Set a new expiration date for specific users
    • Only visible for already enrolled learners
    • Helpful for granting additional time
  8. How to Manage Course Groups
    • Assign or unassign groups to/from courses
    • Group user enrollment is controlled via the Group Admin

  3. Click the blue "Publish" button.
    • It is located at the top-right (as highlighted in the screenshot).
    • If the course is still a draft, publishing it will make it live and accessible to students based on visibility settings.

Course Deletion & Restoration

How to Delete a Course

Method 1

  1. Open the course editor for the course you want to delete.
  2. Look at the top-left corner of the screen.
  3. Click the red "Move to trash" button (highlighted in the image).

‚ö†Ô∏è What Happens Next?

  • The course is not permanently deleted immediately.
  • It is moved to the Trash, where you can later:
    • Restore it, or
    • Permanently delete it from the Trash folder in the courses list.

Method 2

  1. From the WordPress Dashboard, go to Course List (as shown on the left sidebar).
  2. Find the course you want to delete in the list.
  3. Click the Trash Bin icon under the course card (highlighted in red in the screenshot).
  4. Confirm the action if prompted.

‚úÖ Result:

  • The course will be moved to the Trash and can be:
    • Restored later, or
    • Permanently deleted from the ‚ÄúTrash‚Äù tab.

How to Permanently Delete a Course

  1. Go to Course List from the left sidebar.
  2. Click the ‚ÄúTrash‚Äù tab at the top to view trashed courses (highlighted in your image).
  3. Find the course you want to delete permanently.

  5. Confirm the deletion when prompted.

‚ö†Ô∏è Note:
